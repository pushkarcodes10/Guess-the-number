# 🎯 Guess The Number Game

A simple and fun **Guess The Number** game built to practice programming fundamentals like loops, conditionals, random number generation, and user input handling.

---

## 📌 Features

- Random number generation
- User-friendly gameplay
- Hints for higher/lower guesses
- Attempt counter
- Win/Lose conditions
- Beginner-friendly project structure

---

## 🚀 How to Play

1. Run the program
2. Enter your guess
3. The game will tell you if the number is:
   - Too high 📈
   - Too low 📉
4. Keep guessing until you find the correct number 🎉

---

## 🛠️ Technologies Used

- JavaScript
- HTML
- CSS
